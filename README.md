# LeetSRS PWA

A spaced repetition system (SRS) for LeetCode problems - now available as a Progressive Web App!

## Features

- 📚 **Card Review System** - Review LeetCode problems using the proven FSRS (Free Spaced Repetition Scheduler) algorithm
- 📝 **Notes** - Add and manage notes for each problem
- 📊 **Statistics** - Track your progress with detailed charts and statistics
- 🌍 **Multi-language Support** - Available in English and Chinese (Simplified)
- ☁️ **GitHub Gist Sync** - Sync your data across devices using GitHub Gists
- 📱 **PWA Support** - Install as an app on mobile devices
- 🌙 **Dark/Light Theme** - Choose your preferred color scheme

## Usage

Visit: https://qjr1997.github.io/leetsrs_pwa/

## Data Management

- **Export**: Backup your data to a JSON file
- **Import**: Restore from a previous backup
- **GitHub Gist Sync**: Automatically sync across browsers

## Development

This PWA is built with:
- React + TypeScript
- Vite
- Tailwind CSS
- TanStack Query
- Chart.js
- ts-fsrs (FSRS algorithm)
